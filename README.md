# Ludvig's Custom Accordion

A very simple, accessible, and minimal accordion block for WordPress block editor (Gutenberg). Built for clean UI/UX and easy content management.

## Installation

1. Upload the plugin to `/wp-content/plugins` or install via the WordPress admin
2. Activate via the Plugins screen
3. Add the "Ludvig's Custom Accordion" block in the block editor

## Usage

-   Click the block to add accordion items
-   Set title, content, and (optionally) an image for each item
-   Choose if multiple items can be open at once
